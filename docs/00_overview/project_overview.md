# Project Overview

## 1. このプロジェクトの目的

Poker Timer App は、ポーカーサークルや小規模トーナメント運営で使うことを想定したトーナメントタイマーである。
単純なカウントダウンタイマーではなく、**トーナメント構造そのものを編集・保存・再利用できること**を重視している。

本アプリが解決したい課題は次のとおり。

- レベルとブレイクを含むトーナメント構造を明示的に扱いたい
- 構造を毎回手入力せず、プリセットとして保存したい
- 実行中のタイマー表示を安定させたい
- 将来的に機能追加しても破綻しにくい設計にしたい

## 2. 現在の到達点

現時点で実現済み、または設計として確立しているものは次のとおり。

- TournamentStructure を唯一の真実とする構造化
- Level / Break を同列 item として扱う設計
- deterministic timer
- Editor の operation model
- Preset の Save / Load / Rename / Delete
- Snapshot ベースの UI

## 3. 何をやろうとしているか

このプロジェクトが目指しているのは、「一度作った構造を安定して使い回せる実用トーナメントタイマー」である。
短期的には次を重視する。

- タイマー挙動の安定化
- Editor の使いやすさ向上
- Preset 管理の実用化
- ドキュメント整備
- テストしやすい設計の維持

中長期的には次が候補になる。

- 設定画面の詳細化
- 構造 validation
- 音通知
- export / import
- サーバ保存
- 共有や同期

## 4. 非目標

現時点では次は対象外、または未確定である。

- マルチユーザ同期
- 権限制御
- サークル分離
- トーナメント履歴の高度管理
- チップ管理や会計連携

これらは `docs/09_roadmap/roadmap.md` で扱う。
