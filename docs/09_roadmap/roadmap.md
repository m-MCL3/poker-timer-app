# Roadmap

本ディレクトリは将来構想を置く場所であり、**現時点の確定仕様ではない**。

---

## 検討候補

- 設定画面具体化
- 音通知 / カウントダウン音
- structure validation
- preset export / import
- サーバ保存
- マルチユーザ対応
- サークル単位の分離
- ディーラーモード
- 権限管理
- トーナメント履歴
- チップ管理連携

---

## 原則

将来機能を追加する場合でも、次は維持する。

- Domain の純粋性
- item 基準構造
- deterministic timer
- UI と Usecase の責務分離
