# Roadmap

本ディレクトリは将来構想を記録する場所であり、確定仕様ではない。

## 候補

- Settings 詳細化
- 音通知
- structure validation
- preset export / import
- サーバ保存
- マルチユーザ対応
- サークル分離
- ディーラーモード
- 権限管理
- トーナメント履歴
- チップ管理連携

## 原則

将来機能を追加する場合でも、次は維持する。

- item 基準構造
- deterministic timer
- Domain の純粋性
- UI と Usecase の責務分離

## 注意

ここに書いてあっても、`01_application_spec` に昇格していなければ実装対象ではない。
