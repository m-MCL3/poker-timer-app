# Roadmap

本書は将来構想を整理するものであり、
現時点では仕様化されていない。

---

## 検討予定機能

- マルチユーザ対応
- サーバ保存
- プリセット共有
- ディーラーモード
- サークル単位分離
- 役割管理（Admin / Dealer / Player）
- トーナメント履歴
- チップ管理連携
- 通知機能

---

## 設計原則

これらを追加する際も、

- Domain の純粋性を維持
- Port 経由で拡張
- UIにロジックを持ち込まない

を厳守する。
