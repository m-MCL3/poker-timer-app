# Poker Timer App (Web / PWA)

## 概要

Poker Timer App は、ポーカーサークル向けのトーナメントタイマーを  
**Clean Architecture 思想に基づいて設計・実装する Web / PWA アプリケーション** です。

本プロジェクトは現在「設計優先フェーズ」にあり、

- 今後の拡張に耐えるアーキテクチャの確立
- データ構造の明確化
- ビジネスルールの分離
- 永続化方式の差し替え可能設計

を最優先としています。

---

## 現在確定している仕様

詳細は `docs/spec.md` および `docs/editor-spec.md` を参照。

### コア機能

- トーナメントタイマー
- レベル進行管理
- ブラインド構造管理
- Editor によるブラインド編集
- プリセット保存 / 読み込み（LocalStorage 実装）

---

## 設計原則

- 仕様未確定の挙動は実装しない
- ビジネスルールは Usecase 層にのみ存在させる
- UI は表示と入力受付のみ
- 永続化は Port 経由
- タイマー状態の正は常に一箇所
