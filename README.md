# Poker Timer App

Poker Timer App は、ポーカーサークル向けのトーナメントタイマーアプリです。

トーナメントのブラインド構造（レベル・ブレイク）を編集し、タイマーとして実行できます。
サークル内のトーナメント運営を簡単にすることを目的としています。

---

# 主な機能

## Tournament Timer

トーナメントのブラインド構造に基づいてタイマーを実行します。

機能

* Start / Pause / Resume
* 自動レベル遷移
* 手動 Next / Previous
* Break 表示
* 次レベルの情報表示

トーナメント進行に必要な情報を一画面で確認できます。

---

## Structure Editor

トーナメント構造を編集する画面です。

編集できる内容

* Level の追加
* Break の追加
* Level / Break の削除
* レベル時間の変更
* ブラインド設定の変更

トーナメント構造を自由に作成できます。

---

## Preset Management

作成したトーナメント構造はプリセットとして保存できます。

機能

* Preset 保存
* Preset 読み込み
* Preset 名変更
* Preset 削除

サークルでよく使う構造を簡単に再利用できます。

---

# 画面構成

アプリは主に以下の画面で構成されています。

### Timer

トーナメントタイマーを実行する画面。

表示内容

* 残り時間
* 現在レベル
* 次レベル
* ブラインド

---

### Editor

トーナメント構造を編集する画面。

* Level
* Break
* ブラインド設定
* レベル時間

を編集できます。

---

# 使い方

1. Editor でトーナメント構造を作成
2. Preset として保存
3. Timer 画面でトーナメントを実行
